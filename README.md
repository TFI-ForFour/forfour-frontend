# ForFour
### 발달장애청년과 마을을 잇는 참여형 산책 커뮤니티 서비스  

**ForFour**는 발달장애 청년들이 **마을 속에서 함께 걷고, 연결되고, 성장할 수 있도록** 돕는 참여형 산책 커뮤니티 서비스입니다. 사용자는 산책방을 **생성하거나 참여**할 수 있으며, 산책 시작과 종료 시 **QR코드를 스캔하여 활동을 인증**할 수 있습니다. 산책을 통해 쌓인 **총 걸은 거리와 참여 횟수에 따라 배지**를 획득할 수 있어, 지속적인 참여를 유도합니다.<br/>
**ForFour**는 단순한 기록 도구를 넘어, 산책을 매개로 **성인병 예방**, **사회적 고립 완화**, **지역사회 참여 확대**를 지원하는 **웹 기반 상호작용 서비스**입니다.

## 프로젝트 개요 (Project Overview)

- **목적**
    - 이 솔루션은 **발달장애청년과 마을을 산책이라는 활동으로 연결**하여, 일상 속에서 자연스럽게 **성인병을 예방하고 사회적 고립을 완화**하는 것을 목표로 합니다.
    - 산책방 생성·참여, QR코드 기반 산책 시작/종료 기록, 뱃지 보상 시스템을 통해 **지속적인 걷기 습관 형성과 지역사회 참여**를 유도합니다.

- **주요 기능**
    1. **산책방 생성 및 참여**: 사용자는 새로운 산책방을 생성하고, 다른 유저는 메인 페이지에서 산책방 확인 후 들어올 수 있습니다.
    2. **QR코드 기반 산책 인증**: 산책 시작 및 종료 시 QR코드를 스캔하여, 산책 시작 및 종료를 간편하게 할 수 있습니다.
    3. **산책 거리 및 횟수 측정**: 사용자의 누적 산책 거리와 산책 참여 횟수를 자동으로 집계합니다.
    4. **뱃지 시스템**: 특정 거리나 횟수 목표를 달성하면 뱃지를 지급하여, **동기부여와 성취감**을 제공합니다.

### 특징 및 장점 (Features & Benefits)

- **사회적 연결과 고립 완화:**  
    산책방을 통해 발달장애청년, 보호자, 마을 주민이 함께 걷는 경험을 공유하면서  
    자연스럽게 **관계 형성과 대화의 기회**가 늘어나고, 사회적 고립을 완화하는 데 도움을 줍니다.

- **건강한 생활 습관 형성:**  
    정기적인 산책 참여와 누적 거리/횟수 기록을 통해 **꾸준히 걷는 습관**을 만들 수 있으며,  
    성인병 예방과 전반적인 **신체·정신 건강 증진**에 기여합니다.

- **간편한 참여와 높은 접근성:**  
    별도의 복잡한 장비 없이 **QR코드 스캔만으로 산책 시작/종료를 인증**할 수 있어,  
    디지털 환경에 익숙하지 않은 사용자도 쉽게 사용할 수 있습니다.

- **동기부여 중심의 뱃지·기록 시스템:**  
    누적 산책 거리와 횟수에 따라 **뱃지와 성취 기록을 시각적으로 제공**함으로써,  
    사용자가 스스로 목표를 설정하고 도전할 수 있는 **게임화(Gamification) 기반 동기부여**를 제공합니다.

## 기대 효과 (Expected Benefits)

### 영향 (Impact)

- **산책 참여율 및 지속성 증가**  
    산책방과 뱃지 시스템을 통해 사용자가 스스로 목표를 설정하고 도전하게 함으로써,  
    **정기적인 산책 참여율과 장기적인 지속성**을 높이는 데 기여합니다.

- **건강 증진 및 성인병 예방**  
    일상 속 걷기 활동을 자연스럽게 늘려 **비만, 고혈압, 당뇨 등 성인병 예방**에 도움을 주며,  
    규칙적인 신체 활동을 통한 **전반적인 신체·정신 건강 향상**에 긍정적인 영향을 미칩니다.

- **사회적 고립 완화 및 관계 형성**  
    혼자가 아닌 **함께 걷는 경험**을 통해 새로운 사람을 만나고 대화를 나누는 기회를 제공합니다.  
    이를 통해 발달장애청년의 **사회적 고립을 완화**하고, **마을 공동체 내 관계 형성**을 돕습니다.

- **지역사회 기반 포용 환경 조성**  
    마을 주민, 보호자, 발달장애청년이 하나의 산책 커뮤니티 안에서 연결되며,  
    지역사회가 **보다 포용적이고 참여적인 환경**으로 변화하는 데 기여합니다.

### 확장 가능성 (Scalability)

- **지역화폐 연동 보상 시스템**  
    산책 참여도와 활동량에 따라 `지역화폐(예: 모아)`를 보상으로 지급하는 구조로 확장할 수 있습니다.  
    이는 사용자에게 실질적인 동기부여를 제공하며, 지역 경제 활성화에도 기여할 수 있습니다.

- **사용자 맞춤 산책 코스 생성 기능**  
    사용자가 직접 **산책 코스를 생성**하고 공유할 수 있는 기능을 추가하여,  
    다양한 취향과 필요에 맞는 코스를 자유롭게 구성할 수 있습니다.

- **장소 등록 및 커뮤니티 기반 지도 구축**  
    산책 중 방문할 수 있는 **편의 시설, 쉼터, 명소 등을 사용자 또는 지역기관이 등록**하도록 확장하면,  
    모두가 함께 만들어가는 **커뮤니티 기반 산책 지도**가 형성될 수 있습니다.

## 설치 및 실행 방법 (Installation & Execution)
**1.의존성 설치**
```bash
yarn install
```
**2.Yarn SDK 의존성 추가**
에디터 및 개발환경에서 자동 완성, 타입 지원 등을 원활하게 사용하기 위해 Yarn SDK를 설치합니다.
```bash
yarn add @yarnpkg/sdks -D
```
**3.VSCode용 Yarn SDK 설정**
VSCode에서 Yarn PnP 환경을 잘 인식할 수 있도록 SDK를 설정합니다.
```bash
yanr dlx @yarnpkg/sdks vscode
```
**4.개발 서버 실행**
프로젝트를 로컬에서 실행합니다.
```bash
yarn dev
```
**5.브라우저에서 접속**
브라우저를 열고 아래와 같이 로컬 주소로 접속합니다.
```bash
http://localhost:5173
```
